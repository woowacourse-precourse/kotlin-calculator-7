# kotlin-calculator-precourse

## 구현할 기능 목록

### 입력받기

- [x] 처음에 커스텀 구분자가 있는지 판단 (정규식)
    - [x] 있다면 구분자 종류에 추가
    - [x] //구분자\n 이후부터 숫자탐색 시작함
- [x] 구분자기준으로 숫자 골라내기

#### 예외(IllegalArgumentException)

- [x] 쉼표(;) 콜론(:) 커스텀 구분자 이외의 문자가 들어가 있는 경우
- [x] 숫자가 아닌 문자열이 들어간 경우
- [x] 음수가 들어가있는 경우

### 계산

- [x] 숫자들 더하기
