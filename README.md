# kotlin-calculator-precourse

## 문자열 덧셈 계산기

### 구현해야 할 기능
- 문자열을 입력받는 기능
  - 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시키는 기능
- 구분자로 문자열을 분리하는 기능
  - 쉼표(,)로 분리하는 기능
  - 콜론(:)으로 분리하는 기능
  - 커스텀 구분자로 분리하는 기능 (//와 \n 사이에 위치하는 문자)
- 분리한 문자열을 숫자로 변환하는 기능
- 변환한 숫자를 합하는 기능
- 덧셈 결과를 출력하는 기능
