# kotlin-calculator-precourse

## 기능 목록

1. **빈 문자열 입력 처리**
   - 빈 문자열이 입력되면 '0'을 반환한다.
   - 예: `"" => 0`

2. **쉼표(,) 또는 콜론(:)을 구분자로 가지는 문자열 덧셈**
   - 쉼표(,) 또는 콜론(:)을 구분자로 사용하여 숫자를 분리한 후, 각 숫자의 합을 반환한다.
   - 예: `"1,2,3" => 6`, `"1:2:3" => 6`, `"1,2:3" => 6`
   
3. **커스텀 구분자 지정**
   - 문자열 앞부분에 `"//[구분자]\n"` 형식으로 커스텀 구분자를 지정할 수 있으며, 한 개의 문자로 구성된다.
   - 지정된 커스텀 구분자를 사용하여 문자열을 분리하고, 각 숫자의 합을 반환한다.
   - 예: `"//;\n1;2;3" => 6`

4. **잘못된 입력 처리**
   - 양수가 아닌 숫자나 잘못된 값이 입력될 경우, `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료된다.
   - 예: `"//;\n2;-1;1" => IllegalArgumentException 발생`
   



### 요구 사항
- 입력은 구분자와 양수로만 구성된 문자열이다.
- 커스텀 구분자는 한 개의 문자로 구성되며, 기본구분자와 혼용하여 사용할 수 있다.
- 잘못된 값이 입력될 경우 적절한 예외 처리를 한다.
